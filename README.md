# Statistics_For_Data-Analysis
This repository is a comprehensive collection of my learning journey and practical exercises in statistics for data analysis. It covers key concepts, probability, distributions, hypothesis testing, regression, and data visualization, all implemented using Python and Jupyter Notebook.

The goal of this repository is to provide a clear, organized, and hands-on reference for anyone looking to strengthen their foundation in statistics for data analysis, machine learning, or data-driven decision-making.

#Key Features
>Descriptive & inferential statistics
>Probability theory & distributions
>Hypothesis testing (Z-test, T-test, Chi-square, ANOVA)
>Regression analysis & model evaluation
>Data cleaning, visualization, and exploration
>Practical Jupyter Notebook exercises

#Purpose
This repository is meant to serve as a learning portfolio, a reference for statistics concepts, and a demonstration of practical data analysis skills using real datasets.
